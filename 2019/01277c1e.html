设置 npm 镜像源(不然安装很忙)

pi@raspberrypi:~/workdir/hexo $ npm get registry
https://registry.npmjs.org/
pi@raspberrypi:~/workdir/hexo $ npm config set registry http://registry.npm.taobao.org/
pi@raspberrypi:~/workdir/hexo $ npm get registry
http://registry.npm.taobao.org/
